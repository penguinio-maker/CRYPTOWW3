# Tank Game (AI + Online MVP)

## Start The Online Server

1. Open a terminal in the `tank-game` folder
2. Run:

```bash
npm install
npm start
```

The server will run at `ws://localhost:8080`.

## How To Play Online

1. Open `tank_game.html` in two tabs/browsers
2. Press `Play Online` for both players
3. When 2 players are connected, the match starts automatically

The `Play vs AI` mode remains fully working and independent.
